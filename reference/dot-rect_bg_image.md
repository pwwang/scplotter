# Draw a rectangle as the image (background) of a spatial plot and fill with a given color

Draw a rectangle as the image (background) of a spatial plot and fill
with a given color

## Usage

``` r
.rect_bg_image(args, fill)
```

## Arguments

- args:

  A list of arguments (the dot arguments) passed to the spatial plot
  function.

- fill:

  The color to fill the rectangle.

## Value

A ggplot2 layer object that can be added to a ggplot.
