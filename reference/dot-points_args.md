# Get points arguments from the dot-dot-dot arguments in the spatial plot function

Get points arguments from the dot-dot-dot arguments in the spatial plot
function

## Usage

``` r
.points_args(args, ...)
```

## Arguments

- args:

  A list of arguments (the dot arguments) passed to the spatial plot
  function.

- ...:

  Additional arguments that will be parsed.

## Value

A list of points arguments.
